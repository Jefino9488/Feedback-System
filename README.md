# Feedback System

Spring Boot REST API for employee signup/login and peer feedback workflows.

## Documentation

- Developer guide: `HELP.md`
- Swagger UI: `http://localhost:8080/swagger-ui/index.html`
- OpenAPI spec: `http://localhost:8080/v3/api-docs`

## Run

```powershell
.\mvnw.cmd spring-boot:run
```

## Test

```powershell
.\mvnw.cmd test
```
